## 公告编辑
格式为:<br>
- 第一行放置标题（# 标题）
- 第二行放置时间（9999/12/31 23:59）
- 第三行开始为正文（请注意这里需要标题请再次放置！）
<a>---</a>为分割线，用于多个公告的分割
## 网页编辑
0. 打开微软大战代码
1. Ctrl+F
2. 一键替换所有的HKMC云为你的云名称
3. 未使用[HKMCMF-cart](https://github.com/hekuo5310/HKMCMF-order)的请将所有的/order替换为/cart
4. 手动修改556~561
5. 手动修改628~631
6. 手动修改633~635
